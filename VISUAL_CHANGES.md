## Terminal Theme Elements Added

1. **Command Prompts**: Used throughout for headers
   - `$ whoami`
   - `$ cat filename.txt`
   - `$ ls -la directory/`

2. **Monospace Typography**: Courier Prime font everywhere

3. **Terminal Colors**:
   - Background: Dark blue
   - Foreground: Off-white
   - Accent: Yellow-gold
   - Secondary: Cyan
   - Borders: Subtle blue-gray

4. **Card Design**:
   - Bordered cards
   - Slightly transparent backgrounds
   - Hover effects with accent color
   - Rounded corners (small radius)

5. **Interactive Elements**:
   - Buttons styled as terminal commands
   - Links with bracket notation
   - Form inputs with terminal styling

6. **Background**:
   - Animated starfield (200 twinkling stars)
   - Canvas-based animation
   - Smooth 60fps rendering
